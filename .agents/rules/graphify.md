# graphify
Graph at `graphify-out/`.
- **Pre-read**: `graphify-out/GRAPH_REPORT.md` (architecture) or `graphify-out/wiki/index.md` (docs).
- **Navigate**: Use graphify MCP (`query_graph`, `get_node`, `shortest_path`) or CLI (`graphify query`, `path`, `explain`). Prefer over `grep`.
- **Update**: Run `graphify update .` after code changes.
